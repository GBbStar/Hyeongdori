## **YOLO 알고리즘 검출률 검증을 위한 프로토타입 구현**

***



### **프로토타입 구현 내용**

- Kaggle Data set - Dogs and Cats을 이용한 객체 인식 알고리즘 구현
  - **테스트 데이터** : 2000장
  - **훈련용 데이터** : 8000장
- YOLO 알고리즘 인식률 분석
- CNN Model 사용
- Google Colaborator 사용
