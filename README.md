## *YOLO 예시코드 실행 방법
### 1. 설치해야 하는 것
  - Anaconda3
  - Python(3.5 version 이상)
  - Pytorch
  - CUDA (9.0)
  - OpenCV
  
### 2. 가상환경 생성/제거 (CMD창에서 명령어 실행)
  - conda create --name torch
  - conda remove --name torch --all
  
### 3. 가상환경 활성화/비활성화
  - activate torch
  - deactivate
  
### 4. Pytorch 설치
  - conda install -c peterjc123 pytorch cuda90

### 5. OpenCV 설치
  - conda install conda install -c conda-forge opencv
